# X-Ray-Bone-Fracture-Detection-Using-CNN-and-OpenCV
AI-based X-ray bone fracture detection system using CNN and OpenCV. The model analyzes medical X-ray images to classify fractured and normal bones. Built with TensorFlow/Keras and computer vision techniques for automated medical image analysis and fracture detection
